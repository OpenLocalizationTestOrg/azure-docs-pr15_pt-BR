# <a name="azure-storage-client-tools"></a>Ferramentas de cliente de armazenamento do Azure

Os usuários de armazenamento do Azure com frequência desejam poderá exibir/interagir com seus dados usando uma ferramenta de cliente de armazenamento do Azure. Nas tabelas abaixo, podemos listar um número de ferramentas que lhe permitem fazer isso. Podemos colocar um "X" em cada bloco se ele oferece a capacidade de tanto enumerar e/ou acessar abstração de dados. A tabela também mostra se as ferramentas é gratuito ou não. "Avaliação" indica que não há uma avaliação gratuita, mas o produto completo não é gratuito. "Y/N" indica que uma versão está disponível gratuitamente, enquanto uma versão diferente está disponível para compra.

Somente fornecemos um instantâneo das ferramentas de cliente de armazenamento do Azure disponíveis. Essas ferramentas podem continuar a evoluir e crescer na funcionalidade. Se houver atualizações ou correções, deixe um comentário para nos informar. O mesmo é verdadeiro que se você souber de ferramentas que devem ser aqui - seria nós felizes adicioná-las.
