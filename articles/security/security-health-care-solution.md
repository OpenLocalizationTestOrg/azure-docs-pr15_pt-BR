# <a name="a-practical-guide-to-designing-secure-health-care-solutions-in-azure"></a>Um guia prático à criação de soluções de saúde seguras no Azure

Iniciantes do setor de saúde, SIs (integradores), fornecedores de software independentes (ISVs) e organizações de saúde considerando uma mudança para o Azure procuram orientações ajudá-los a incorporam controles de segurança para atender às suas obrigações de conformidade.

[A guia prático para criação de soluções de saúde Secure no Microsoft Azure](https://aka.ms/azureindustrysecurity) ajuda você a entender como você pode melhorar a segurança de suas soluções, usando os serviços do Azure e os recursos que podem ser configuradas com base nas suas necessidades.
O conteúdo é dividido em três seções principais:

1. Orientação de considerações para usar a tecnologia de nuvem, incluindo gerenciamento de risco, responsabilidade compartilhada, estabelecer um sistema de gerenciamento de segurança de informações, setor de compreensão e regulamentos locais e estabelecer procedimentos operacionais padrão.
2. Princípios de segurança de chave que estão alinhados a um padrão de gerenciamento de segurança de informações padrão, como ISO 27001 e processos de desenvolvimento padrão, como da Microsoft do ciclo de vida SDL (Security Development).
3. Aplicando os princípios fundamentais para usar casos demonstrando o alinhamento de uma perspectiva de arquitetura de solução, onde os requisitos para as soluções são alinhados com o gerenciamento de segurança de informações padrão.

Esperamos que você encontre [Um guia prático para criação de soluções de saúde Secure](https://aka.ms/azureindustrysecurity) úteis e se você tiver perguntas ou sugestões, fale conosco, deixando um comentário abaixo.
