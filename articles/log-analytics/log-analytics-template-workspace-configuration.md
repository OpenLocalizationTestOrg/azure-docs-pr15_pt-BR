# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Gerenciar a análise de Log usando modelos do Gerenciador de recursos do Azure

Você pode usar [modelos do Gerenciador de recursos do Azure] (... / azure-resource-manager/resource-group-authoring-templates.md) para criar e configurar espaços de trabalho de análise de Log. Exemplos de tarefas que você pode executar com modelos incluem:

+ Criar um espaço de trabalho
+ Adicionar uma solução
+ Criar pesquisas salvas
+ Criar um grupo de computador
+ Habilitar a coleção de logs do IIS de computadores com o Windows agent instalado
+ Coletar contadores de desempenho de computadores Linux e Windows
+ Coletar eventos de syslog em computadores Linux 
+ Coletar eventos de logs de eventos do Windows
+ Coletar logs de eventos personalizados
+ Adicionar o agente de análise de log para uma máquina virtual Azure
+ Configurar a análise de log para dados de índice coletados usando o diagnóstico do Azure


Este artigo fornece exemplos que ilustram parte da configuração que você pode executar a partir de modelos de um modelo.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Criar e configurar um espaço de trabalho de análise de Log

O modelo de exemplo a seguir ilustra como:

1.  Criar um espaço de trabalho

5.  Habilitar a coleção de logs do IIS de computadores com o Windows agent instalado

### <a name="deploying-the-sample-template"></a>Implantando o modelo de exemplo

Para implantar o modelo de exemplo:

1. Salvar a amostra anexada em um arquivo, por exemplo`azuredeploy.json` 
2. Editar o modelo para que a configuração desejada
3. Usar o PowerShell ou da linha de comando para implantar o modelo

#### <a name="powershell"></a>PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Linha de comando

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
